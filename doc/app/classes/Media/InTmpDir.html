Module Media::InTmpDir

  • lib/media/in_tmp_dir.rb

Provides methods useful dealing with temporary folders:

  • #in_tmp_dir takes a block in which you can access to a temporary folder path set to @tmp_dir

  • #tmp_path returns the supplied path relative to @tmp_dir

Example

class VideoProcessing include Media::InTmpDir def process puts @tmp_path #=> nil #in_tmp_dir do puts @tmp_path #=> “/tmp/rrgiun39fsf9jwnwq3r” puts #tmp_path('video.mp4') #=> “/tmp/rrgiun39fsf9jwnwq3r/video.mp4” end puts @tmp_path #=> nil end end

in_tmp_dir() Link

Takes a block in which you can access to a temporary folder path set to @tmp_dir. It takes care of clearing @tmp_dir after the block execution or if an error occurs

Source: show 

# File lib/media/in_tmp_dir.rb, line 27
def in_tmp_dir
  Dir.mktmpdir(Rails.application.config.tempfiles_prefix.call) do |dir|
    @tmp_dir = dir
    yield
  end
ensure
  @tmp_dir = nil
end

def tmp_path(path)
  raise Error.new('@tmp_dir must be present', path: path) unless @tmp_dir
  File.join(@tmp_dir, path)
end
