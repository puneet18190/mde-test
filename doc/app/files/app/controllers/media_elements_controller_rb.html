Description

Controller for all the actions related to elements and their buttons. All over this controller we use the constant keywords defined in ButtonDestinations, namely: 1. found_media_element (or simply found) for a media element seen in a results list of the search engine 2. compact_media_element (or simply compact) for a media element seen in the compact mode 3. expanded_media_element (or simply expanded) for a media element seen in expanded mode (this happens only in the dashboard, see DashboardController)
